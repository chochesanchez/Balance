# Balance - User Stories
# Free Plan & Pro Plan | Ages 15-50

> **Goal:** Build the finance app people can't live without.
> Every story is designed for **engagement, retention, and monetization**.

---

## How to Read This Document

Each user story follows this format:

> **As a** [persona], **I want to** [action], **so that** [outcome].

Stories are tagged with:
- **Retention Hook** - Why they keep coming back
- **Monetization Lever** - How this drives Pro upgrades
- **Delight Factor** - What makes them tell their friends

---

# FREE PLAN - 20 User Stories

> The free plan must be **genuinely useful** - not a crippled demo.
> Users should love the app for months before hitting a natural limit that makes Pro feel like an obvious upgrade.

## FREE-01: Effortless First Transaction

**As a** 15-year-old who just received birthday money,
**I want to** record my first income in under 10 seconds with a beautiful calculator keypad,
**so that** I feel an instant sense of control over my money and want to keep tracking.

**Acceptance Criteria:**
- Custom calculator-style number pad (no system keyboard)
- One-tap type selection (Income / Expense / Transfer)
- Auto-suggested title based on amount and time of day
- Celebratory confetti animation on first-ever transaction
- Haptic feedback on every tap
- "Your first step to financial freedom!" toast message

**Retention Hook:** The first 30 seconds determine if they keep the app. Make it magical.
**Delight Factor:** Confetti + haptics + encouraging message = screenshot-worthy moment.

## FREE-02: Smart Onboarding in 60 Seconds

**As a** 22-year-old college student downloading the app between classes,
**I want to** set up Balance in under a minute with my name, currency auto-detected, and one quick goal selected,
**so that** I start using the app immediately without a tedious setup process.

**Acceptance Criteria:**
- 4 screens max: Welcome > Name + Currency > Goal > Ready
- Currency auto-detected from device locale (with override option)
- Skip button always visible (no forced fields)
- Animated transitions between steps (spring physics)
- Progress bar shows completion
- No email, phone, or profile photo required
- "You can customize everything later" reassurance text

**Retention Hook:** Zero friction = no drop-off. They're inside the app in 60 seconds.
**Delight Factor:** "Wow, that was fast" - compared to every banking app they've used.

# PRO PLAN - 20 User Stories

> Pro features must feel like **superpowers** - not basic features held hostage.
> Each Pro story should make free users think "I NEED that."

---

## PRO-01: Apple Pay Transaction Import

**As a** 22-year-old who pays for everything with Apple Pay,
**I want to** automatically import my Apple Pay transactions into Balance with one tap,
**so that** I never have to manually enter purchases I've already made.

**Acceptance Criteria:**
- FinanceKit integration (iOS 17+) with user permission
- Import screen: list of unrecorded Apple Pay transactions
- Each transaction shows: amount, merchant name, date
- Smart category matching: "Starbucks" → Food, "Uber" → Transport
- One-tap import or "Import All" button
- Duplicate detection (won't re-import existing transactions)
- Account auto-assignment to "Apple Pay" account
- Background refresh option: auto-import daily (with notification)

**Retention Hook:** This is the killer feature. Zero-effort tracking = perfect data = better insights.
**Monetization Lever:** The #1 reason to upgrade. "Never manually enter a purchase again."

---

## PRO-02: Multi-Currency with Live Conversion

**As a** 30-year-old digital nomad earning in USD but spending in EUR and MXN,
**I want to** track transactions in multiple currencies and see totals converted to my base currency,
**so that** I understand my true financial position regardless of which country I'm in.

**Acceptance Criteria:**
- Assign currency per account (Cash EUR, Bank USD, etc.)
- Record transactions in any currency
- Dashboard shows total balance in base currency
- Live exchange rates (updated daily)
- Historical rate preservation (transaction keeps the rate at time of recording)
- Currency conversion calculator built in
- Conversion shown inline: "€45.00 (~$48.50)"
- Up to 5 simultaneous currencies

**Retention Hook:** No other free finance app handles multi-currency well. This is their home.
**Monetization Lever:** Travelers and remote workers will pay $3/mo for this without blinking.

## PRO-03: iCloud Sync Across All Devices

**As a** 50-year-old who uses both iPhone and iPad,
**I want to** have all my financial data synced seamlessly across devices via iCloud,
**so that** I can record on my phone and review on my iPad with the same data.

**Acceptance Criteria:**
- CloudKit private database (user's own iCloud)
- Real-time sync with conflict resolution
- Sync indicator in Settings: "Last synced: 2 min ago"
- Manual "Sync Now" button
- Graceful offline handling: queue changes, sync when connected
- No Balance server involved (privacy preserved)
- Data encrypted in transit and at rest
- Works across iPhone, iPad, and future Mac app

**Retention Hook:** Once synced across devices, migration cost is too high to switch apps.
**Monetization Lever:** Multi-device users are power users willing to pay.
